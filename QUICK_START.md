## 🛠️ TỰ SETUP (Cho kỹ thuật viên)

### Yêu cầu
- Node.js 18+
- Git
- Vercel CLI: `npm i -g vercel`
- GitHub account

### Steps

```bash
# 1. Clone repo
git clone https://github.com/dinhkhanhtung/tranhhangkhoa.git
cd tranhhangkhoa

# 2. Install dependencies
npm install

# 3. Setup Firebase (tạo project mới)
# Copy .env.example → .env.local

# 4. Chọn ngành
# Edit: src/context/WebsiteContext.tsx
# Change: defaultSettings.industry = "your-industry"

# 5. Run locally
npm run dev

# 6. Deploy
vercel --prod
```
